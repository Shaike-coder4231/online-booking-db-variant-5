# Онлайн-запись в салон красоты с выбором мастера и дополнительных услуг

**Выполнил:** Баженов Илья
**Группа:** 454
**Вариант:** 5
**СУБД:** MySQL 8.0+
**Кодировка:** utf8mb4_unicode_ci

## Описание предметной области

Система предназначена для автоматизации процесса онлайн-записи клиентов в салон красоты. Реализована возможность выбора конкретного мастера, дополнительных услуг и расходных материалов. Некоторые услуги требуют обязательной предварительной консультации. При создании записи система проверяет наличие необходимых продуктов на складе.

## Стек технологий

- СУБД: MySQL 8.0 или выше
- Клиент: MySQL Workbench / DBeaver / консольный клиент
- Формат схемы: SQL (DDL/DML)
- Документация: Markdown / PDF
- Контроль версий: Git + GitHub

## Структура репозитория

В корне проекта расположен файл README с описанием работы. Папка sql содержит скрипт создания схемы базы данных schema.sql и файл проверки ограничений constraints_test.sql. В папке docs хранится отчёт по проекту в формате Markdown или PDF, а также подпапка images для скриншотов выполнения запросов. Папка queries включает отдельные файлы с примерами запросов к базе данных. В папке diagrams размещена ER-диаграмма схемы базы данных. Файл .gitignore настроен для исключения временных и системных файлов.
